**RN 34**
=========
O campo Unidades para Disponibilização deve conter os mesmos Ministérios indicados no campo Coautores quando da criação do processo (excluído o Ministério Autor). Deve vir pré-preenchido e não ser passível de alteração. A alteração deve ser permitida apenas no campo Coautores do processo, sendo refletida no campo Unidades para Disponibilização em questão.
